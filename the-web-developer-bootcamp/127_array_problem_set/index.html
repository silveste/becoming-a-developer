Array Problem set

To test this exercise using the console it's required to create an array and call the functions described below pasing the array as argument

  • printReverse(array): Print the array in reverse mode
  • isUniform(array): Returns true if all values in the array are the same value, if not returns false
  • sumArray(array): It assumes all values in the array are numbers and returns the sum of them
  • max(array): It assumes all values in the array are numbers and returns the max value of them
